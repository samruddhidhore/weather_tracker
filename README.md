# India Weather App
A beautiful and modern web application that displays real-time weather
and tourist information for cities across India. .

# Live Weather Updates
-   Temperature\
-   Feels like\
-   Humidity\
-   Wind speed\
-   Weather description & icon

# City Information
-   Famous spot of the selected city\
-   Short description\
-   City + State details\
-   Dynamic background image based on the city

# Project Structure
    /project-folder
    │── index2.html      # Main webpage
    │── style.css        # UI styling and animations
    │── script.js        # Logic for weather API & city list
    │── README.md        # Project documentation

# Setup Weather API
In `script.js`, update your API key:
``` js
const apiKey = "YOUR_API_KEY";
```
Get a free key from: https://openweathermap.org/api
